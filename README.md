# EthGuess Monorepo

EthGuess is a price-prediction dApp built with Next.js and NestJS.

## Project Structure

- `apps/frontend`: Next.js application
- `apps/backend`: NestJS application (to be initialized)

## Getting Started

This project uses `pnpm` for package management and workspaces.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [pnpm](https://pnpm.io/) (v8+)

### Setup

```bash
# Install dependencies
pnpm install
```

### Development

```bash
# Run all apps in development mode
pnpm dev
```
